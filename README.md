# TypeScript Plugin for DocPad
Adds support for [TypeScript](http://Typescriptlang.org/) to JavaScript compilation to [DocPad](https://docpad.org)

Convention:  `.js.ts`


## Install

```
npm install --save docpad-plugin-typescript
```
